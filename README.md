##关于

这是一份关于VIM插件管理器Vundle的帮助文档的中文翻译

##安装

```
$ git clone https://maucat@github.com/maucat/vundle.doc.git ~/.vim/doc/
```
然后在vim里输入
```
:helptags ~/.vim/doc

如果你已经在使用Vundle插件，那么

```
$ git clone https://maucat@github.com/maucat/vundle.doc.git ~/.vim/bundle/vundle/doc/ 
```
然后在vim里输入
```
:BundleInstall
```
至此你就可以使用亲爱的中文帮助文档了！
